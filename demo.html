Kernel Methods

  • Idea: learn linear models in a feature space induced by a kernel function.
  • Feature space is usually high-dimensional or even of infinite dimension. So, it is usually not explicitly calculated.
  • Methods: Support Vector Machines, kernel ridge regression, kernel pca, and many more (nowadays called shallow methods)
  • Problem: efficiency \(O(N^2)\)

Problem Statement (1/2)

  • Anomaly and outlier detection is an important task in different research areas with various application domains.
  • One of the main challenges in anomaly detection is that usually only normal data points are available.
  • Density-based algorithms such as parametric density estimation or kernel density estimation can be used to solve anomaly and outlier detection problems.
  • Current density estimation algorithms suffer from the curse of dimensionality and their linear complexity in the number of training data is prohibitive inmost application domains.
